Template Installation

  1. FTP Upload:
    • Open up your FTP manager and connect to your hosting
    • Browse to required directory (Normally public_html)
    • Upload the files inside jantrik folder.

Pages

    • index.html is for Home page
    • index-2.html is for Home Two page
    • index-3.html is for Home Three page
    • shop.html is for Shop page
    • compare.html is for best compare product page
    • cart.html is for Cart page
    • checkout.html is for Checkout page
    • wishlist.html is for Wishlist page
    • blog.html is for Blog page
    • blog-details.html is for Blog Detail page
    • contact.html is for Contact page
    • product.html is for product page
    • register.html is for registration page
    • login.html is for login page
    • forgot-password.html is for forgot password page
    • about-us.html is for about us page
    • 404.html is for 404 error page
    • account.html is for customer account page
